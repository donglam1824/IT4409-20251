Introduction

Python is used in many application domains.

print("Hello, Python")
Web and Internet Development

Python offers many choices for web development:

  • Django
  • Flask
  • Pyramid
  • Web2Py

Python's standard library supports many Internet protocols:

  • HTTP
  • FTP
  • SMTP
  • POP3

And the Package Index has yet more libraries:

  • Requests
  • BeautifulSoup
  • Selenium
  • Scrapy
pip install flask
Scientific and Numeric

Python is widely used in scientific and numeric computing. Some of the popular libraries are:

  • NumPy
  • Pandas
  • Matplotlib
  • Scikit-learn
import numpy as np
Education

Python is a great language for teaching programming, both at the introductory level and in more advanced

courses. 
for i in range(3): print(i)
Desktop GUIs

The Tk GUI library is included with most binary distributions of Python.

Some toolkits that are usable on several platforms are available separately:

  • WxPython
  • PyQt
  • PyGTK

Platform-specific toolkits are also available:

  • win32ui
  • MacPython
import tkinter as tk
Software Development

Python is often used as a support language for software developers, for build control and management, testing, and in many other ways.

Business Applications

Python is used in many business applications, including ERP and e-commerce systems.
